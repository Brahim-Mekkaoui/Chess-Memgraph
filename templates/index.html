    🔬 Puissance du Graphe — Exemples Cypher

    Ces requêtes illustrent pourquoi Memgraph (base graphe) est idéale pour modéliser les relations entre ouvertures d'échecs.

    // Variantes de la Sicilienne
    MATCH (o:Opening) WHERE o.eco_code STARTS WITH 'B' RETURN o.name, o.eco_code ORDER BY o.eco_code
    // Ouvertures similaires
    MATCH (o:Opening {eco_code:'C97'}) -[:SIMILAR_TO]-(s) RETURN o.name AS source, s.name AS similaire
    // Hubs du graphe
    MATCH (o:Opening) -[r:SIMILAR_TO]-() RETURN o.name, count(r) AS deg ORDER BY deg DESC LIMIT 10
